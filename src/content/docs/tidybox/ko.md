## 1. TidyBox란?

TidyBox는 다운로드 폴더나 바탕화면에 쌓인 파일들을 **규칙에 따라 자동으로 분류**해주는 앱입니다.

### 이런 분께 추천합니다

- 다운로드 폴더가 항상 지저분한 분
- 파일 정리에 시간 쓰기 싫은 분
- 파일을 체계적으로 관리하고 싶은 분

### 기본 사용 흐름

```
1. 정리할 폴더 선택 (예: 다운로드)
2. 규칙 설정 (예: PDF는 문서 폴더로)
3. Clean 버튼 클릭
4. 끝! 파일이 자동으로 분류됩니다
```

## 2. 처음 시작하기

앱을 처음 실행하면 설정 마법사가 자동으로 시작됩니다.

### 1단계: 환영 화면

**Get Started** 버튼을 눌러 시작하세요.

### 2단계: 정리할 폴더 선택

- **Downloads** — 다운로드 폴더 (기본 선택)
- **Desktop** — 바탕화면
- **Add Custom Folder...** — 원하는 폴더 직접 선택

> 💡 여러 폴더를 동시에 선택할 수 있습니다. **Scan Folders** 버튼을 눌러 다음 단계로 진행하세요.

### 3단계: 프리셋 선택

미리 만들어진 규칙 모음을 선택하세요:

- **General** — 일반 사용자용 (이미지, 문서, 동영상 등)
- **Developer** — 개발자용 (코드, 설정 파일 등)
- **Designer** — 디자이너용 (PSD, Sketch 등)

> 💡 프리셋을 선택하지 않고 **Skip** 버튼으로 건너뛸 수도 있습니다. 선택한 경우 **Apply & Continue** 버튼을 눌러 진행하세요. 나중에 설정에서 언제든 변경할 수 있습니다.

### 4단계: 스캔 결과

선택한 폴더에 있는 파일 현황을 보여줍니다. **Continue** 버튼을 눌러 다음 단계로 진행하세요.

> 💡 모든 단계에서 **Back** 버튼을 눌러 이전 단계로 돌아갈 수 있습니다.

### 5단계: 요금제 선택

- **14일 무료 체험** — 모든 기능을 무료로 체험
- **무료 플랜으로 시작** — 기본 기능만 사용

## 3. 대시보드 사용하기

대시보드는 TidyBox의 메인 화면입니다.

### 대시보드 열기

- 메뉴바 아이콘 클릭 → **Open TidyBox** 선택
- 또는 단축키 `⌘O`

### 화면 구성

| 영역 | 설명 |
|------|------|
| **상단 통계** | 대기 중인 파일 수, 오늘 정리한 파일 수, 마지막 정리 시각 |
| **파일 목록** | 정리 대상 파일들과 이동 예정 경로 |
| **하단 버튼** | Clean Now, Undo, Activity, Settings |

### 파일 미리보기

파일 옆의 **눈 아이콘**을 클릭하면 파일 내용을 미리 볼 수 있습니다.

### 경고 아이콘 안내

| 아이콘 | 의미 |
|--------|------|
| ⚠️ | 대용량 파일 (500MB 이상) |
| 🔒 | 접근 권한이 없거나 사용 중인 파일 |
| ☁️ | iCloud에만 있는 파일 (로컬에 없음) |
| 📄📄 | 같은 이름의 파일이 이미 존재 |

## 4. 메뉴바 사용하기

TidyBox는 메뉴바에 항상 표시됩니다.

### 메뉴 항목

| 메뉴 | 단축키 | 설명 |
|------|--------|------|
| Open TidyBox | `⌘O` | 대시보드 열기 |
| Clean Now | `⌘K` | 즉시 정리 실행 |
| Undo Last Action | `⌘Z` | 마지막 작업 되돌리기 |
| Manage Rules... | `⌘,` | 규칙 관리 화면 열기 |
| Activity Log... | `⌘L` | 활동 기록 보기 |
| Settings... | — | 설정 열기 |
| Quit TidyBox | `⌘Q` | 앱 종료 |

## 5. 규칙 만들기

규칙은 "어떤 파일을 어디로 이동할지" 정하는 것입니다.

### 규칙 추가하기

1. **Settings** → **Rules** 탭 선택
2. **+** 버튼 클릭
3. 규칙 이름 입력
4. 조건 설정 (예: 확장자가 pdf)
5. 액션 선택 (예: Documents 폴더로 이동)
6. **Save** 클릭

### 사용 가능한 조건

#### 기본 조건 (Free)

| 조건 | 설명 | 예시 |
|------|------|------|
| 파일 확장자 | 특정 확장자 파일만 | png, jpg, pdf |
| 파일명 포함 | 이름에 특정 단어 포함 | Screenshot |
| 파일 크기 | 특정 크기 이상/이하 | 100MB 이상 |
| 파일 유형 | 이미지, 동영상 등 카테고리 | 이미지, 문서 |

#### 고급 조건 (Pro)

| 조건 | 설명 | 예시 |
|------|------|------|
| 정규식 패턴 | 복잡한 이름 패턴 매칭 | report_2024 |
| 생성일 | 특정 날짜 이전/이후 | 2024년 1월 이전 |
| 수정일 | 마지막 수정 날짜 기준 | 최근 7일 |
| 파일 경과일 | 생성된 지 N일 이상/이하 | 30일 이상 된 파일 |
| 중복 파일 | 내용이 같은 파일 감지 | — |

### 사용 가능한 액션

| 액션 | 설명 |
|------|------|
| **폴더로 이동** | 지정한 폴더로 이동 |
| **날짜 폴더로 이동** | 연/월별 폴더를 자동 생성하여 이동 |
| **격리** (Pro) | 임시 보관 후 자동 삭제 |
| **제외** | 이 규칙에서 제외 (건드리지 않음) |

### 규칙 순서

- 위에 있는 규칙이 먼저 적용됩니다
- 드래그하여 순서를 변경할 수 있습니다

## 6. 프리셋 사용하기

프리셋은 미리 만들어진 규칙 모음입니다.

### General 프리셋

일반 사용자를 위한 기본 규칙:

- 설치 파일 (dmg, pkg) → Installers 폴더
- 압축 파일 (zip, rar) → Archives 폴더
- 이미지 → Images/연-월 폴더
- 문서 (pdf, docx) → Documents 폴더
- 동영상/음악 → Media/연-월 폴더
- 스크린샷 → Screenshots/연-월 폴더
- 대용량 파일 (500MB+) → Large Files 폴더

### Developer 프리셋

개발자를 위한 규칙:

- 코드 파일 → Dev/Code 폴더
- 설정 파일 (json, yaml) → Dev/Config 폴더
- 로그 파일 → Dev/Logs/연-월 폴더
- 데이터베이스 파일 → Dev/Database 폴더
- 셸 스크립트 → Dev/Scripts 폴더

### Designer 프리셋

디자이너를 위한 규칙:

- 디자인 소스 (psd, sketch, fig) → Design/Sources 폴더
- 내보낸 파일 (png, svg) → Design/Exports/연-월 폴더
- 폰트 파일 → Design/Fonts 폴더
- 3D/모션 파일 → Design/3D & Motion/연-월 폴더

### 프리셋 적용하기

1. **Settings** → **Rules** 탭
2. **Load Preset** 버튼 클릭
3. 원하는 프리셋 선택
4. 규칙이 자동으로 추가됩니다

> 💡 기존 규칙은 유지되며, 새 규칙만 추가됩니다.

## 7. 파일 정리하기

### 정리 실행하기

1. 대시보드에서 정리할 파일 목록 확인
2. **Clean Now** 버튼 클릭
3. 확인 창에서 **Clean** 선택

### 또는 메뉴바에서

- 메뉴바 아이콘 클릭 → **Clean Now** (`⌘K`)

### 정리 결과

정리가 완료되면:
- 시스템 알림으로 결과 안내
- 대시보드에서 상세 결과 확인 가능

### 파일 충돌 시

같은 이름의 파일이 이미 있으면 자동으로 번호를 붙입니다:
```
report.pdf → report (1).pdf → report (2).pdf
```

## 8. 되돌리기

실수로 잘못 정리했다면 되돌릴 수 있습니다.

### 되돌리기 방법

- **대시보드** → **Undo** 버튼
- **메뉴바** → **Undo Last Action** (`⌘Z`)
- **활동 기록** → 개별 파일 되돌리기

### 되돌리기 기간

| 요금제 | 기간 |
|--------|------|
| Free | 24시간 |
| Pro / Trial | 30일 |

### 되돌리기가 안 되는 경우

- 파일이 삭제되었거나 이동된 경우
- 파일 내용이 변경된 경우
- 되돌리기 기간이 지난 경우

---

## 9. 활동 기록 보기

지금까지의 모든 정리 기록을 확인할 수 있습니다.

### 활동 기록 열기

- **메뉴바** → **Activity Log...** (`⌘L`)
- **대시보드** → **Activity History** 버튼

### 기록 확인하기

각 정리 기록에서 확인할 수 있는 정보:
- 정리 일시
- 이동된 파일 수
- 실패한 파일 수
- 어떤 방식으로 실행되었는지 (수동/자동/예약)

### 개별 파일 되돌리기

기록을 펼치면 개별 파일별로 되돌리기 버튼이 표시됩니다.

### CSV 내보내기 (Pro)

활동 기록을 CSV 파일로 저장할 수 있습니다:
- 오늘 / 최근 7일 / 최근 30일 / 전체 기간 선택

## 10. 자동 정리 설정하기

Pro 버전에서는 자동으로 파일을 정리할 수 있습니다.

### 예약 정리

설정한 시간마다 자동으로 정리합니다.

**설정 방법:**
1. **Settings** → **General** 탭
2. **Scheduled Cleaning** 켜기
3. 주기 선택 (15분 / 30분 / 1시간 / 2시간 / 4시간 / 하루 1회)

### 방해금지 시간

특정 시간대에는 자동 정리를 멈출 수 있습니다.

**설정 방법:**
1. **Quiet Hours** 켜기
2. 시작/종료 시간 설정 (예: 22:00 ~ 08:00)

### 배터리 옵션

- **배터리 사용 시 건너뛰기** — 충전 중일 때만 자동 정리
- **배터리 부족 시 건너뛰기** — 20% 미만일 때 건너뛰기

### 파일 감시

새 파일이 생성되면 자동으로 정리합니다.

**설정 방법:**
1. **Settings** → **General** 탭
2. **File Watcher** 켜기

> 💡 새 파일이 다운로드되면 2초 후 자동으로 정리됩니다.

### 시작 시 자동 실행

앱이 시작될 때 자동으로 정리를 실행합니다.

**설정 방법:**
1. **Settings** → **General** 탭
2. **Run on Startup** 켜기

## 11. 격리 기능 사용하기

격리는 파일을 바로 삭제하지 않고 임시 보관하는 기능입니다.

### 격리 규칙 만들기

1. 규칙 편집에서 액션을 **Move to Quarantine** 선택
2. 보관 기간 설정 (7일 / 30일 / 90일)
3. 만료 후 처리 방법 선택:
   - **휴지통으로 이동** — 안전하게 삭제
   - **영구 삭제** — 완전히 삭제 (복구 불가)
   - **계속 보관** — 삭제하지 않고 유지

### 격리 파일 관리하기

1. **Settings** → **General** 탭
2. **Quarantine** 섹션에서 **Manage Files...** 클릭

### 관리 화면에서 할 수 있는 것

- **검색** — 파일명으로 찾기
- **복원** — 원래 위치로 되돌리기
- **남은 일수 확인** — 녹색(여유), 주황(곧 만료), 빨강(만료 임박)

### 격리 폴더 비우기

모든 격리 파일을 삭제하려면:
1. **Settings** → **General** 탭
2. **Empty Quarantine...** 클릭

> ⚠️ 이 작업은 되돌릴 수 없습니다.

## 12. 설정

### General (일반)

| 설정 | 설명 |
|------|------|
| Launch at Login | Mac 시작 시 자동 실행 (Pro) |
| Show in Dock | Dock에 아이콘 표시 |
| Notifications | 알림 설정 |
| Scheduled Cleaning | 예약 정리 설정 (Pro) |
| Quarantine | 격리 파일 관리 |

### Folders (폴더)

정리할 폴더를 추가하거나 제거합니다.
- **+** 버튼으로 폴더 추가
- **-** 버튼으로 폴더 제거

### Rules (규칙)

규칙을 만들고 관리합니다.
- 규칙 추가/편집/삭제
- 프리셋 불러오기
- 제외 패턴 설정

### Account (계정)

현재 요금제 확인 및 구매 관리.

## 13. 요금제 안내

### 무료 플랜

- 규칙 최대 2개
- 기본 조건만 사용 가능
- 되돌리기 24시간

### Pro 플랜 ($9.99, 1회 결제)

- 규칙 무제한
- 모든 조건 사용 가능
- 되돌리기 30일
- 예약 정리
- 파일 감시
- 격리 기능
- CSV 내보내기
- 로그인 시 자동 시작

### 14일 무료 체험

- 카드 등록 없이 시작
- 14일간 Pro 기능 전체 사용
- 만료 후 자동으로 무료 플랜으로 전환

### 구매 방법

1. **Settings** → **Account** 탭
2. **Upgrade to Pro** 클릭
3. App Store에서 결제

### 구매 복원

다른 Mac에서 이미 구매했다면:
1. **Settings** → **Account** 탭
2. **Restore Purchases** 클릭

## 자주 묻는 질문

### Q: 파일이 정리되지 않아요

- 규칙이 활성화되어 있는지 확인하세요
- 해당 파일이 규칙 조건에 맞는지 확인하세요
- 🔒 아이콘이 있다면 접근 권한 문제입니다

### Q: 되돌리기가 안 돼요

- 되돌리기 기간이 지났는지 확인하세요 (Free: 24시간, Pro: 30일)
- 파일이 이동되거나 삭제되지 않았는지 확인하세요

### Q: 폴더 접근 권한이 필요하다고 해요

시스템 환경설정 → 보안 및 개인 정보 → 전체 디스크 접근 권한에서 TidyBox를 허용해주세요.

### Q: 자동 정리가 작동하지 않아요

- Pro 또는 Trial 상태인지 확인하세요
- Quiet Hours 시간대가 아닌지 확인하세요
- 배터리 옵션 설정을 확인하세요
